# DNYF TETCH Admin Panel

This is the admin panel for the DNYF TETCH website.

## Access
- URL: `https://dnyftetch.github.io/dnyf-tetch-ai-website/admin/`
- Default credentials: `example-admin` / `example123`

## Features
- Dashboard with statistics
- Content management
- Analytics
- User management
- Settings and backup

## Security
1. Change default password immediately
2. Access via Ctrl+Shift+A on main site
3. Data stored in browser localStorage
